PROJECT_01 · ZOHO CRM AUTOMATION
Zoho CRM Quotation Automation System
Standard Roofs, Erode · March 2026 · Solo Build · 95% Complete · Production Live
// Overview

A complete end-to-end CRM automation system that replaced a 20-minute manual quoting process with a one-click, sub-2-minute automated flow. Built entirely solo within the first 2 months of joining as AI Engineer — with no prior Zoho CRM experience.

The system automates the entire lifecycle from lead capture to professional PDF quote delivery, covering 25+ of 30+ fields automatically, managing product images via Cloudinary CDN, and storing final PDFs in WorkDrive with permanent links in both Quote and Deal records.

// System Architecture — 4 Automation Layers
Layer 1
Workflow Rules (14+)
Trigger Deluge functions on CRM record create/edit events across web and Zoho mobile app
Layer 2
Deluge Functions
Backend logic — auto-fetch client data, roofing specs, staff contacts, product images, serial numbers
Layer 3
Client Script (JavaScript)
Browser-side instant auto-fill — Total Area and Grand Total calculate live as user types, before save
Layer 4
API Connections (Zoho Writer)
One-click PDF generation via mergeAndStore API — auto-attaches to CRM record, stores in WorkDrive
// Data Flow
01
LEAD — Pre-sales enquiry captured in Zoho Leads module
02
CONTACT — Permanent customer data store, single source of truth for client info
03
DEAL — Active roofing project record; feeds into Quote and stores final PDF link
04
QUOTE — Transaction module where all 4 automation layers converge; user enters only 5 fields
05
ZOHO WRITER — Mail merge template with 30+ placeholders auto-filled; rendered as professional PDF
06
PDF → CRM STORAGE — Auto-stored in WorkDrive, attached to Deal, permanent link saved in both records
// Key Automation Functions Built

Create Fetch Function — Auto-populates client name, contact, site location from linked Contact record on quote creation.

Roofing Autofill (Master Spec Fetch) — Fetches complete roofing system specifications from Product master records — covering 8 products and 65+ subtypes, all automated.

Smart Image Logic — Priority system: if user manually uploads an image, it takes precedence; otherwise Cloudinary CDN URL auto-fetched from Product Colors module.

PDF Generation Function — Calls Zoho Writer mergeAndStore API, generates named PDF with smart filename (Quote No + Client Name), auto-attaches to CRM Deal.
