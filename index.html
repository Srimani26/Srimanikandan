PROJECT_01 · ZOHO CRM AUTOMATION
Zoho CRM Quotation Automation System
Standard Roofs, Erode · March 2026 · Solo Build · Production Live
// Overview

A complete end-to-end CRM automation system that replaced a 20-minute manual quoting process with a one-click, sub-2-minute automated flow. Built entirely solo within the first 2 months of joining — with no prior Zoho CRM experience.

The system automates the entire lifecycle from lead capture to professional PDF quote delivery. Out of 30+ fields in the Quotes module, the user only needs to enter 5 — the automation stack handles everything else.

// 4 Automation Layers
Layer 1
Workflow Rules (14+)
Trigger Deluge functions on CRM record create/edit — works on both web and Zoho mobile app
Layer 2
Deluge Functions
Backend logic — auto-fetch client data, roofing specs, staff contacts, product images, serial numbers
Layer 3
Client Script (JavaScript)
Browser-side instant auto-fill — Total Area and Grand Total calculate live as user types, before save
Layer 4
Zoho Writer API
One-click PDF via mergeAndStore API — auto-attaches to CRM record, stores in WorkDrive permanently
// End-to-End Data Flow
LEAD
Pre-sales enquiry captured in Zoho Leads module
CONTACT
Permanent customer data store — single source of truth for client info
DEAL
Active roofing project record — feeds into Quote and stores final PDF link
QUOTE
Transaction module — all 4 automation layers converge here. User enters only 5 fields
WRITER
Zoho Writer mail merge — 30+ placeholders auto-filled, rendered as professional PDF
STORE
Auto-stored in WorkDrive + attached to Deal + permanent link saved in both records
// Key Automation Functions

Create Fetch Function — Auto-populates client name, contact, and site location from linked Contact record the moment a quote is created.

Roofing Autofill (Master Spec Fetch) — Fetches complete roofing system specifications — covering 8 products and 65+ subtypes, all automated.

Smart Image Priority Logic — If user manually uploads an image, it takes precedence. Otherwise, Cloudinary CDN URL auto-fetched from Product Colors module.

PDF Generation Function — Calls Zoho Writer mergeAndStore API, generates a smart-named PDF (Quote No + Client Name). Under 10 seconds end-to-end.

Staff Auto-Fetch — Supervisor, Manager, and Accountant mobile numbers auto-fetched and embedded in every quote automatically.
